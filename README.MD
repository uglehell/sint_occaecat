# Site design and layout "Sint Occaecat"

## Design made in Figma

6 sections:
  * HOME (intro)
  * RISQUIS (content)
  * GALLERY (swiper)
  * EMPTY (parallax)
  * NOSTRUD (content)
  * CONTACT (info and links)

## Layout

technologies used:
  * PUG index
  * SCSS styles
  * JavaScript without jQuery
  * GULP 4 build

characteristics:
  * Adaptive
  * Cross Browser
  * BEM
  * Semantics
  * CSS animations (transition, keyframes)
